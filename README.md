# swift-quic-examples
Example project for QUIC using Network.framework.

<img src="docs/screenshot.png" width="320">

## Client
### Requirements
* Xcode 14.3+
* iOS 16+ or macOS 13+

### Build
Open xcodeproj and build swift-quic-examples.

```
$ open swift-quic-examples.xcodeproj
```

## Server
### Requirements
* Go 1.19+

### Start Echo Server

```
$ make echo
```
